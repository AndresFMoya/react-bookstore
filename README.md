### Install
- To get started with the app, first clone the repo and `cd` into the directory:
  
  ```
  $ git clone https://github.com/AndresFMoya/react-bookstore.git
  $ cd react-bookstore
  ```
- `npm install`

- Start the app locallly:

- `npm start`
